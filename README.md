# Expense Tracker Android App

## 📱 Features
- Add Expenses
- Add Income
- Track Total Spending
- Track Total Income
- Calculate Remaining Budget

## 🛠 Tech Stack
- Kotlin
- XML Layouts
- SharedPreferences (Local Storage)
